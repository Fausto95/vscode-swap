## VS CODE SWAP 🔁

<b>Swap 🔁</b> is a VS Code extension that makes swapping texts easy 😎

<img src=".github/assets/Swap.gif" width="1100px" height="500px" style="border-radius: 4px"/>


### Instructions

* Select two texts
* Open `command palette` and type `Swap`
* Press enter and voilà ✅
